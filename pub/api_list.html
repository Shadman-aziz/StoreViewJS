Getting Started (setup)

make sure to include the following lines in the header of your HTML file

 
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">
  <link rel="stylesheet" href="card-make.css">
  <script src="dom_load.js"> </script>
  <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"> </script>
  <script defer type="text/javascript" src='javascript-library.js'> </script>

this is how you declare a StoreViewJS card collection 

 
  const collection = new CardCollection()

this is how you make a card collection object

 
  collection.makeCollection(classToAdd,withCartFunctionality,clickToEnlarge,wishListFunctionality)

this is how you add a card object to your collection object

 
  collection.makeCard(name,imgFolder,numOfPics,title,content,variableValue,price,description)

Options

all the options available for each Collection object

Parameter Default Type Description
classToAdd (required) / String you pass the class attribute given to the HTML element (preferably a div element) that you want the card collection to be added to (please make sure that this class attribute is unique). For example, if the div element that you want to add to has the unique class attribute "FirstContainer" then you will pass in ".FirstContainer" as the value for this parameter
withCartFunctionality False String you declare whether you want the "add to cart" functionality for your card collection, meaning the feature where if you hover over the card you get the "add to cart" option. If you want to enable it pass "True" for this parameter
clickToEnlarge False String you declare whether you want the "click To enlarge" functionality for your card collection, meaning the feature where if you click the image in the card you get the pop-up window with the extended description and the option to click between pictures of the catalog of the product. If you want to enable it pass "True" for this parameter
wishListFunctionality False String you declare whether you want the "wish list" functionality for your card collection, meaning there will be a small "heart" sign in middle of cards of this collection which if you click the card gets added to the wishlist for the particular collection.If you want to enable it pass "True" for this parameter

all the options available for each Card object

Property Type Description
name String name you want to give the product in this particular card
imgFolder String (path to folder of images) for this card, going to be animating when user is hovering over particular card. Example : "images/shoe3_pics/" (please make sure to follow this naming syntax for all the pictures in the folder "nameOfCard"+numberStartingFrom1+".jpeg" example: if card name is Greece the names of the pictues will be Greece1.jpeg,Greece2.jpeg etc)
numOfPics integer the number of pictures the imgFolder has or how many pics you want to be used for the on-hover animation for the card. If you dont want hover animation please enter 1. Make sure not to enter a number larger than the number of files in the imgFolder
title String the title you want to be shown in the card for the item/product
content String the short description you want to be shown in the card for the item/product
variableValue integer the number of likes the item/product has
price integer or float the price that will show up in the card for the item/product
description String the long description you want to be shown for the item/product in the pop-up window when the image is clicked ("clickToEnlarge" property of the particular collection needs to be enabled for this)

Functions

Functions available for each card collection

  //make collection of cards
const collection = new CardCollection()
collection.makeCollection("classToAdd","withCartFunctionality","clickToEnlarge","wishListFunctionality")

//add card to collection
 collection.makeCard("name","imgFolder",numOfPics,"title","content",variableValue,price,"description")

//sort by price
 collection.sortByPrice()

//how to switch cards
 collection.switchPosition(positionOfCardToSwitch1,positionOfCardToSwitch2)

//how to show liked cards
  collection.showLiked()

//how to edit title for a particular card
  collection.editTitle(numberOfTheCardToEdit,newPrice)

//how to edit price for a particular card
  collection.editPrice(numberOfTheCardToEdit,newPrice)

//how to edit content for a particular card

  collection.editContent(numberOfTheCardToEdit,newPrice)

//how to highlight card
  collection.highlightCard(numberOfTheCardToHighlight)

//how to remove highlight from card
  collection.removeHighlightCard(numberOfTheCardToRemoveHighlight)

//how to show cart, when submit is pressed inside cart a JSON object with customer information is returned
  collection.checkoutCart()

//how to show wishlist
  collection.showWishList()

//how to delete particular card
  collection.deleteCard(numberOfTheCardToDelete)
